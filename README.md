# AI-Assisted Business Data Analysis

## Project overview
This project simulates real-world work of a junior data analyst in a business environment.
The analysis was conducted end-to-end using SQL, Power BI and an AI-assisted workflow.

## Business context
A medium-sized company operating in a competitive market aims to better understand its sales performance and customer behavior.

## Objectives
- Define key business questions and KPIs
- Analyze data using SQL
- Identify trends and insights
- Build a Power BI dashboard
- Provide actionable business recommendations

## Tools & technologies
- SQL
- Power BI
- Python / Excel
- AI-assisted analysis workflow

## Project structure
The repository contains:
- Business problem definition
- Data understanding and assumptions
- SQL analysis
- Analytical insights
- Dashboard
- Final recommendations
