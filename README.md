# Business Data Analysis Portfolio

End-to-end business data analysis project using SQL and Power BI, focused on real-world analytical tasks, insights, and business decision-making.

## Project Overview
This project simulates the real-world work of a junior data analyst in a business environment.  
The analysis was conducted end-to-end using SQL for data analysis and Power BI for visualization and insights delivery.

## Business Context
A medium-sized company operating in a competitive market aims to better understand its sales performance and customer behavior in order to support data-driven decision-making.

## Objectives
- Define key business questions and KPIs  
- Analyze data using SQL  
- Identify trends and patterns in performance  
- Build an interactive Power BI dashboard  
- Provide actionable business recommendations  

## Tools & Technologies
- SQL (SQLite)
- Power BI
- Excel / Python (supporting analysis)
- AI-assisted workflow (analysis support)

## Project Structure
The repository contains:
- Business problem definition
- Data understanding and assumptions
- SQL analysis
- Analytical insights
- Power BI dashboard
- Final business recommendations
